# Administrator guide

Sign in with the configured administrator account. Run discovery only after setting
credentials, cost limits, domain policy, and the live-crawl flag. Review each
candidate's original URL, publisher, robots outcome, score breakdown, duplicate
group, rights, and directness; approval creates a source record but does not certify
its extracted text.

After crawling, confirm the immutable snapshot and extraction strategy, compare raw
and cleaned versions, correct metadata/speakers, merge duplicates, select the
primary underlying work, review semantic chunks, and enable only supported chunks.
Then embed, test search debug ranks, run evaluations, and inspect failed/refused
answers. Use cost and job views for retries, cancellations, and budget monitoring.
All significant changes must have an audit event.
